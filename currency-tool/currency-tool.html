Currency Tool

CurrencyTool is an android application which returns a list of currency values relative to the selected denomination. It is somewhat similar to a standard currency conversion app.

In order to retrieve the necessary information, the application uses the DOM parser to parse through an XML sheet containing the values of over 30 major denominaitons. The XML file is published and updated daily by the European Central Bank.

Source code for this app is available on my Github page.
